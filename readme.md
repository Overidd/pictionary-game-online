

## Ejecutar el proyecto

```
   1: npm run install
   2: coviar el archivo .env.template a .env
   3: npm run dev
```

## Rutas existentes 
 ```
   /login -> El usuario ingresa su nombre de usuario
 ```
 ``` 
   /room -> El usuario busca una sala
 ```
 ```
   /game -> El usuario ingresa al juego canvas
 ```
